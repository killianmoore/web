# App Updates For Members And Vendors

Quick edit files:

- Members CSV: `members-2024.csv`
- Vendors CSV: `vendors-2024.csv`
- Full playbook: `pd-release-runbook-v1.md`

Member building/address edits:

- Open `members-2024.csv`
- Find the member row
- Edit `Work Address`, `Apt. #`, `City`, `State`, or `Zip`

Vendor category edits:

- Open `vendors-2024.csv`
- Add the vendor once per category
- Do not duplicate the same vendor twice inside the same category

After editing the CSV files, update the website. A new TestFlight build is not needed for CSV-only member/vendor changes because the iOS app loads the live PD website.
